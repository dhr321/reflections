When would you want to use a remote repository rather than keeping all your work
local?

    Using a remote repository is preferable when you use multiple computers or if you're collaboratin with other people.

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    If you pull automatically, then whatever you pull down might overwrite something that you've been working on.

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?

    Fork is a copy of a repository on GitHub. Clones are copies of repostitories from GitHub to local or local to local. Branches are copies of commits within a repository.
